npmプロジェクトのライブラリバージョンアップメモ

2023/04/29 15:29

毎回忘れるのでメモ

npm-check-updates[1]をグローバルにインストール

$ npm i -g npm-check-updates

npm outdatedで状況確認する

$ npm outdated

ncuコマンドで、マイナーバージョンのアップデートを行う

$ ncu -u --target minor

npm ciではなくnpm installする

$ npm install

npm lsで一応現在のライブラリのバージョンを確認しておくと良い

$ npm ls

おわり。

脆弱性のあれこれ

  • npm audit
  • npm audit fix --force
脚注

  1. raineorshine/npm-check-updates: Find newer versions of package dependencies than what your package.json allows ↩︎
